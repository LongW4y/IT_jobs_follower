# IT job offers follower

### Supported job portals:

NoFluffJobs

JustJoinIT

#### Jobs portals planned to support:

- BulldogJob,
- SolidJobs

#### How to use:

Add label and url with filters to [offers/job_offers.yaml](offers/job_offers.yaml) and run the script periodically.
